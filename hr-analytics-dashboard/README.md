# 👥 Workforce Attrition & Risk Analytics Dashboard

## 📌 Project Overview

This project analyzes employee data to understand attrition trends, identify high-risk employees, and provide actionable HR insights using Power BI, SQL, and Python.

---

## ❓ Problem Statement

Employee attrition is a major challenge for organizations. This project aims to:

* Identify departments with high attrition
* Detect high-risk employees
* Understand factors influencing employee turnover

---

## 🛠️ Tools & Technologies

* Power BI
* SQL
* Python (Pandas, Matplotlib)

---

## 📊 Dashboard Features

* Attrition Analysis (by department, role)
* Risk Profiling (High, Medium, Low)
* KPI Cards (Total Employees, Attrition Rate)
* Interactive Filters

---

## 🔍 Key Insights

* Certain departments show significantly higher attrition
* Employees with lower tenure tend to leave more
* Risk segmentation helps identify employees likely to leave

---


## 📁 Project Structure

* data/ → Dataset
* dashboard/ → Power BI file
* sql/ → SQL queries
* python/ → Analysis notebook


---

## 🚀 How to Use

1. Download the `.pbix` file from dashboard folder
2. Open in Power BI Desktop
3. Use filters to explore insights
