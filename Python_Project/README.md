
# Global Superstore Data Analysis (Python)

## 📌 Objective
To perform data cleaning, exploratory data analysis (EDA), and visualization on sales data using Python to uncover business insights.

---

## 📊 Analysis Performed

### 🔹 Data Cleaning
- Handled missing values
- Converted data types (dates, numeric)
- Removed duplicates and inconsistencies

### 🔹 Exploratory Data Analysis (EDA)
- Checked distribution of sales and profit
- Identified top-performing categories and regions
- Analyzed customer behavior

### 🔹 Data Visualization
- Monthly sales trend analysis
- Category-wise sales comparison
- Profit distribution
- Region-wise performance

---

## 🛠 Tools & Libraries Used
- Python
- Pandas
- Matplotlib

---

## 📁 Dataset
- Global Superstore dataset  
(Contains information about orders, customers, products, and sales)

---

## 📌 Key Insights
- Sales show an increasing trend over time
- Technology category contributes highest revenue
- Certain regions dominate overall sales performance
- A few customers generate a large portion of total revenue

---

## 🚀 Skills Demonstrated
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization
- Analytical Thinking using Python
