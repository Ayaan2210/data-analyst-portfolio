# 🏏 IPL Performance & Match Intelligence Dashboard

## 📌 Project Overview

This project analyzes IPL (Indian Premier League) data to uncover insights about team performance, player statistics, and match outcomes using SQL, Python, and Power BI.

---

## ❓ Problem Statement

Cricket analytics is essential for understanding team strategies and player performance. This project aims to:

* Identify top-performing teams and players
* Analyze the impact of toss decisions
* Discover match-winning patterns

---

## 🛠️ Tools & Technologies

* SQL (Data Cleaning & Querying)
* Python (Pandas, Matplotlib)
* Power BI (Dashboard & Visualization)

---

## 📊 Dashboard Features

* KPI Cards (Total Matches, Seasons, Top Team, Top Scorer)
* Top Teams by Wins
* Toss Impact Analysis
* Top Run Scorers
* Top Wicket Takers
* Interactive Filters (Season, Team)

---

## 🔍 Key Insights

* Certain teams dominate across multiple seasons
* Toss decisions influence match outcomes significantly
* A few players consistently lead in runs and wickets

---


## 📁 Project Structure


* dashboard/ → Power BI file
* sql/ → SQL queries
* python/ → Analysis notebook


---

## 🚀 How to Use

1. Open `.pbix` file in Power BI Desktop
2. Use slicers to filter by season/team
3. Explore insights interactively
